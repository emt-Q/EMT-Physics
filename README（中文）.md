# EMT Physics 物理探索网站

## 项目简介
EMT Physics 是一个面向物理初学者的交互式学习平台，致力于为更多学习者提供平等的物理教育机会。平台通过AI图像识别技术自动分析物理题目关键词，精准匹配对应的PhET交互式物理仿真实验，让抽象的物理概念变得直观可感，激发学习者的探索兴趣。

## 核心功能
✨ **沉浸式物理美学界面**
- 毛玻璃质感导航栏，跟随物理运动曲线的交互动画
- 品牌光谱渐变配色，模拟能量流动的视觉效果
- Canvas动态量子粒子背景，营造物理宇宙氛围
- 全页面弹簧阻尼过渡动画，还原真实物理运动质感

⌨️ **动态交互体验**
- 首页使命宣言打字机效果，模拟信号传输的光标动画
- 探究者姓名个性化问候，带微交互反馈
- 实验启动倒计时爆炸动画、能量脉冲按钮、原生Web Audio生成的启动音效
- 匹配成功后的随机物理主题激励语录

🔍 **AI智能实验匹配**
- 基于Tesseract.js的中英文OCR文字识别，支持上传物理题目图片自动解析
- 内置关键词匹配引擎，目前支持：
  - 单摆实验室（匹配关键词：单摆、摆长、周期、简谐、pendulum、period等）
  - 电路模拟器（匹配关键词：电阻、电流、电压、欧姆、circuit、resistance等）
- 一键跳转至PhET官方交互式仿真实验室，无需额外安装软件

📱 **全设备适配**
- 响应式布局，完美适配桌面端与移动端浏览器
- 移动端优化的触控交互与字体大小

## 技术栈
- **纯静态实现**：单HTML文件，内嵌CSS样式与原生JavaScript逻辑，无需构建工具
- **OCR引擎**：Tesseract.js v5（CDN加载，支持中英文识别）
- **仿真实验**：PhET Interactive Simulations 开源物理仿真平台
- **音效生成**：Web Audio API 原生合成启动音效，无需外部音频文件
- **背景动画**：Canvas 2D 实现粒子物理运动效果

## 使用方法
1. 环境要求：现代浏览器（Chrome 90+、Edge 90+、Safari 15+），需保持网络连接（用于加载OCR引擎与打开仿真实验页面）
2. 直接在浏览器中打开 `index.html` 文件即可使用
3. 使用流程：
   1. 在输入框填写你的探究者姓名，点击「继续」获得个性化问候
   2. 点击上传区域，选择包含物理题目的本地图片（支持单摆、电路相关题型）
   3. 等待OCR识别进度完成，识别结果将显示在右侧面板
   4. 匹配成功后将显示对应实验名称与激励语录，点击「开启物理实验」
   5. 体验3秒实验启动动画后，将自动在新标签页打开PhET仿真实验室

## 扩展说明
如需添加更多物理实验类型，可在 `index.html` 的JavaScript代码中修改 `PHYSICS_SIMS` 数组：
```javascript
const PHYSICS_SIMS = [
    { 
        name: '实验名称 / English Name', 
        keywords: ['关键词1', '关键词2', 'english keyword'], 
        url: 'PhET实验链接' 
    },
    // 继续添加更多实验
];
```

## 注意事项
- OCR识别准确率与图片清晰度、文字对比度相关，建议使用文字清晰、无过多背景干扰的题目图片
- 实验页面跳转至PhET官方平台，需保持网络连接才能正常加载仿真内容
- 首次使用OCR功能时，浏览器会自动下载中英文识别语言包，可能需要几秒等待时间
